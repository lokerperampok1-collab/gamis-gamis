# Reference Assets

Folder ini berisi aset-aset referensi yang digunakan sebagai panduan dalam pengembangan proyek ini.

## Isi Folder
- **Halaman Statis**: Template HTML untuk berbagai halaman (login, register, contact, dll).
- **Assets**: File pendukung seperti CSS, JavaScript (vendor), dan folder `_nuxt`.

## Catatan Penting
- Folder ini **hanya untuk referensi**.
- Jangan melakukan perubahan langsung pada file di folder ini.
- Gunakan file-file di sini sebagai acuan struktur dan desain saat mengimplementasikan fitur di folder utama aplikasi.
